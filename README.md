Library Management UI:
1. Aim
    To build a library management system with book search, add, and remove functionality.

2. Objective
    ● Create responsive book listing
    ● Implement search functionality
    ● Add form for new book entries
    ● Enable book removal
    ● Manage state with React hooks

3. Hardware/Software Requirements
    ● Software: Node.js 18+, React 18+, VS Code
    ● Hardware: i5+ CPU, 8GB RAM

4. About the Program
    Demonstrates CRUD operations, state management with hooks, and form handling in React.
